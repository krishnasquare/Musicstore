# 🎸 Hope Music House – Tirupati

A modern, premium dark-themed website for **Hope Music House**, a musical instrument store in Tirupati, Andhra Pradesh.

---

## ✨ Features

- **Three.js** animated rotating guitar in the hero section
- **Framer Motion** page & scroll animations throughout
- **Dark premium theme** — black + gold + neon accents
- **Glassmorphism cards** with hover effects
- Fully **responsive** (mobile-first)
- **Sticky navbar** with mobile hamburger menu
- **Floating WhatsApp** button with pulse animation
- **Floating Call** button (mobile only)
- Product grid with **category filter**
- **Google Maps** embed in contact section
- **SEO optimized** meta tags + JSON-LD schema

---

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm 9+

### Install & Run

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🌐 Deploy to Vercel

1. Push this folder to a **GitHub repository**
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your GitHub repo
4. Framework: **Vite** (auto-detected)
5. Click **Deploy** ✅
